# Planarity-Warpage-Analysis-Tool
平面翘曲分析工具
平面翘曲分析工具
这是一个基于 Web 的平面度与翘曲分析工具，支持多种数据格式输入，提供直观的 3D 可视化与工程指标计算。完全在浏览器中运行，无需安装任何软件。

✨ 功能特性
多种数据输入

矩阵格式（直接粘贴 10×10 或任意行列的数值矩阵）

原始格式（带有 #*# 标记的测量数据，自动提取）

核心工程指标

平面度（ISO 1101 最小区域法）

最大翘曲量（原始数据峰谷差）

最大/最小/平均高度

标准差

3D 可视化

基于 Three.js 的交互式曲面图

支持旋转、缩放、网格/表面切换

自动旋转模式

热力图

颜色映射高度值，直观显示平面高低分布

历史记录

自动保存最近 100 次分析结果

一键加载历史数据

报告导出

将当前分析结果导出为文本文件

🚀 快速开始
在浏览器中打开 index.html（现代浏览器均可，建议 Chrome / Firefox / Edge）。

页面默认加载一组示例数据，可直接查看分析结果。

如需分析自己的数据：

在左侧“输入数据”区域选择格式（矩阵/原始）。

粘贴数据，点击“计算分析”。

右侧 3D 图和指标将自动更新。

使用右侧控制按钮调整视图或导出报告。

📊 数据格式说明
矩阵格式
每行数据用空格或逗号分隔，行数即为 Y 方向点数，列数即为 X 方向点数。例如：

text
-0.457, -0.312, -0.145, -0.008
-0.525, -0.344, -0.150, -0.027
...
原始格式
支持类似以下格式（以 #*# 开头的行会被自动过滤）：

text
#*# points =
#*#   -0.457, -0.312, -0.145, -0.008
#*#   -0.525, -0.344, -0.150, -0.027
...
注意：所有数值应为实数，行列数应一致（工具会自动补齐）。

🧮 算法说明
平面度（最小区域法）
采用多起点优化（BFGS 变体 + 梯度下降）寻找一对平行平面，使其恰好包容所有测量点，两平面间的最小距离即为平面度，符合 ISO 1101 标准。

翘曲量
工程中常指原始数据的最大值与最小值之差，即峰谷值。

其他统计量
平均值、标准差、最大最小值均基于原始数据直接计算。

⚠️ 注意事项
平面度计算基于简化优化算法，对于非常极端或噪声较大的数据，结果可能与专业计量软件略有差异，但工程应用足够准确。

若数据点数量较多（如超过 30×30），计算可能略有延迟，但现代浏览器通常可流畅处理。

历史记录仅保存在当前会话中，刷新页面会清空（但可通过“导出报告”保存数据）。
